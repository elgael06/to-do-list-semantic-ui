# To-do list

## Descricion de aplicacion web.

la siguiente aplicacion es un ejemplo de una lista donde los datos se guardan en una base de datos no relacional y como conexion utiliza un servidor escrito en express.

## Deploy:
La aplicacion se desplego en la plataforma de heroku en la url [todo list]('http://todolist.mya-app.tech') .

## Ejecucion en desarrollo local.

Para ejecutar la aplicacion puede seguir los comoandos escritos en el archivo `makefile`.

### Ejemplo :

-   ``` make server``` :
    Este comando ejecuta la aplicacion de servidor por default el puerto 8080.

- ``` make front``` :
    Para ejecutaer el front en modo desarrollo en el puerto 3000.

- ``` make build ``` :
    Para crear el build donde se almacenan los archivos necesarios para mostrar en produccion.
